# leaflet-rs

A [wasm-bindgen](https://github.com/rustwasm/wasm-bindgen)
wrapper for
[Leaflet.js](https://leafletjs.com/)

## Usage

```toml
[dependencies]
leaflet = "0.1"
```

## Example

See https://github.com/kartevonmorgen/leaflet-rs/tree/master/src/example
